# 대형광고물 200m 반경 판정 도구 - 실행 가이드

---

## STEP 1. Python 설치 (처음 한 번만)

1. 아래 주소를 브라우저 주소창에 입력해서 접속
   ```
   https://www.python.org/downloads/
   ```

2. **Download Python** 버튼 클릭

3. 설치 파일 실행 후 **반드시 아래 항목 체크**

   ```
   ☑ Add Python to PATH   ← 이거 꼭 체크해야 합니다!
   ```

   ![설치 화면에서 Add Python to PATH 체크](https://www.python.org)

4. **Install Now** 클릭 → 설치 완료

5. 설치 확인: 명령 프롬프트(cmd)에서 아래 입력
   ```
   python --version
   ```
   → `Python 3.x.x` 라고 나오면 성공

---

## STEP 2. 파일 다운로드 및 압축 해제

1. 메일에 첨부된 **`대형광고물판정도구_no_bat.zip`** 저장
2. 저장된 zip 파일 **우클릭 → 압축 풀기** (또는 더블클릭 후 파일 꺼내기)
3. 압축 해제하면 **`dist-standalone`** 폴더 생성됨

---

## STEP 3. 서버 실행

1. `dist-standalone` 폴더를 탐색기에서 열기

2. 폴더 상단 **주소창 클릭** 후 `cmd` 입력하고 엔터

   ```
   [주소창에 cmd 입력]
   C:\Users\홍길동\Downloads\dist-standalone  →  cmd  →  엔터
   ```

3. 검은 창(명령 프롬프트)이 열리면 아래 명령어 입력 후 엔터

   ```
   python -m http.server 8080
   ```

4. 아래 메시지가 나오면 서버 실행 성공
   ```
   Serving HTTP on 0.0.0.0 port 8080 ...
   ```

---

## STEP 4. 브라우저에서 접속

1. 크롬, 엣지 등 브라우저 열기
2. 주소창에 아래 입력 후 엔터

   ```
   http://localhost:8080
   ```

3. 지도 화면이 뜨면 완료!

---

## STEP 5. 사용 방법

1. 상단 검색창에 **주소 입력** (예: 서울특별시 강남구 테헤란로 123)
2. **검색** 버튼 클릭
3. 지도에 200m 반경 원이 표시됨
4. 반경 내 기존 대형광고물 위치 확인

---

## 종료 방법

- 명령 프롬프트 창에서 **Ctrl + C** 누르기
- 또는 명령 프롬프트 창 닫기

---

## 문제 해결

| 증상 | 해결 방법 |
|------|-----------|
| `python`이 인식 안 됨 | Python 재설치, "Add to PATH" 체크 확인 |
| 지도가 안 뜸 | 인터넷 연결 확인 |
| 디렉토리 목록이 뜸 | dist-standalone 폴더 안에서 cmd 실행했는지 확인 |
| 포트 충돌 오류 | `8080` 대신 `8081` 로 변경: `python -m http.server 8081` |
