대형광고물 200m 반경 판정 도구 - 사용법
==========================================

[실행 방법]

방법 1: Python이 설치된 경우 (권장)
  1. 이 폴더(dist-standalone)를 바탕화면이나 원하는 위치에 복사
  2. 폴더 안에서 주소창에 "cmd" 입력하여 명령 프롬프트 열기
  3. 아래 명령어 입력:
       python -m http.server 8080
  4. 브라우저에서 http://localhost:8080 접속

방법 2: Python 3인 경우
       python3 -m http.server 8080

[종료 방법]
  명령 프롬프트에서 Ctrl+C 누르기

[주의]
  - 브라우저에서 직접 index.html을 더블클릭하면 지도/검색이 작동하지 않습니다
  - 반드시 http://localhost:8080 으로 접속해야 합니다
